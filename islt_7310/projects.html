Projects

Web Design

For many of my web design projects, I have used either Figma or Adobe XD to create the initial high-fidelity mockups. For the MidwayUSA Checkout specifically, I used Adobe XD and then coded the design in VSCode. The new Midway logo was created in a program called Affinity Designer. I like all of the projects in this section that I have created. There are a few things here and there that I would improve on, but overall I am happy with the work I produced.

products

The only product I have currently made and worked on is a program called BlockLoader. It is a Minecraft mod manager which is currently being developed to help streamline the modding process. It is going to be completely free to use and be supported as long as the game is updated. It has access to all of the games' mods and is able to keep track of mod updates and game versions for the user. I really enjoyed working on this project. It has been such a great experience being able to work on a project from the very beginning of concept creation up until the finished project.

Illustrations

These illustrations have been my favorite thing to work on. Some of them are from my fall semester at Mizzou, but others are for real clients. I have a lot of fun creating illustrations. They let me be extremely creative and imaginative. The mountain illustration in particular was a joy to work on. It really let me explore my limits. All of the work in this section was created in Affinity Designer. An alternative to Illustrator. In my opinion, Affinity Designer is a better program than Illustrator.
